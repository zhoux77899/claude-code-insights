# Claude Code Plugins Insights

This directory contains automated daily insights for the Claude Code plugins.

## Files

| File | Description |
|------|-------------|
| `cached-repos.txt` | Cached repositories list |
| `plugins-daily-insight.md` | Daily report with top repositories and trends |
| `repos.json` | Structured JSON data of indexed repositories |

## File Details

### cached-repos.txt

Contains the cached Claude Code plugin repositories.

### repos.json

Contains the complete dataset of indexed Claude Code plugin repositories.

**Data Structure:**

```json
{
  "total_count": 899,
  "items": [
    {
      "id": 574523116,
      "name": "awesome-chatgpt-prompts",
      "full_name": "f/awesome-chatgpt-prompts",
      "html_url": "https://github.com/f/awesome-chatgpt-prompts",
      "description": "A curated list of ChatGPT prompts",
      "stargazers_count": 143195,
      "forks_count": 18993,
      "pushed_at": "2026-01-22T03:37:30Z"
    }
  ]
}
```

**Field Descriptions:**

| Field | Description |
|-------|-------------|
| `total_count` | Total number of indexed repositories |
| `items` | Array of repository objects |
| `id` | GitHub repository ID |
| `name` | Repository name |
| `full_name` | Owner/repo format (e.g., "user/repo") |
| `html_url` | Direct link to the repository |
| `description` | Repository description from GitHub |
| `stargazers_count` | Number of stars |
| `forks_count` | Number of forks |
| `pushed_at` | Last push timestamp (ISO 8601) |

### plugins-daily-insight.md

Daily generated Markdown report containing:

- **Header**: Generation timestamp and total indexed count
- **Top 30 Repositories**: Table with rank, name, stars, forks, last updated, and description
- **Active Repositories**: List of repositories updated in the last 7 days (top 15)

## Manual Trigger

To manually trigger the workflow:

1. Go to the repository's Actions tab
2. Select "Claude Code Plugins Daily Insight" workflow
3. Click "Run workflow"

## Data Source

Repositories are discovered by searching GitHub for files named `marketplace.json` in the `.claude-plugin/` directory path.

```
https://api.github.com/search/code?q=filename:marketplace.json+path:.claude-plugin
```
